# My notes app

## Local development

### Node version

- 10.16.0

### Set up

- Replace `.env.temp` with `.env`
- Run `npm install`
- Add ui settings to firebase

### Run

- `npm start`

### Local Firebase deployment

- `firebase deploy`

## Prod

- Obtain firebase token with `firebase login:ci`
- Add env variables `REACT_APP_PROJECT_ID` and `FIREBASE_TOKEN`
